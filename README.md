# 📈 Stock Portfolio Tracker

A modern fintech-style dashboard for tracking stock investments, analyzing portfolio performance, and visualizing financial insights using interactive charts.

This project helps users monitor their investments, calculate profit/loss, and understand portfolio allocation through a clean and intuitive analytics interface.

---

## 🚀 Features

### 📊 Portfolio Analytics
- Track multiple stock investments
- Calculate total investment value
- Monitor current portfolio value
- Analyze profit and loss in real time

### 📈 Data Visualization
- Portfolio allocation chart
- Profit/Loss bar chart
- Interactive financial dashboard

### 💹 Investment Tracking
- Add stock symbol
- Enter quantity and buy price
- View investment performance instantly
- Delete stocks from portfolio

### 🎨 Premium Dashboard UI
- Modern fintech-style design
- Gradient background
- Responsive card layout
- Clean data tables
- Interactive charts


in your browser.

1. Add stock information:

```

Quantity: 10
Buy Price: ₹150
```

- investment value

- profit/loss
- portfolio allocation

3. Charts will visualize the portfolio performance.
